Introdução

                

                
Estrutura URL de rastreamento de conversão..

            


            

                Nesta página
            

            

                
O pixel é uma imagem que se integra na página de confirmação de uma compra ou registro, ou em plataformas, como o Google Tag Manager, que possuem campos específicos para adicionar tag(pixel). Este pixel é acionado por cookies, que são
                    salvos automaticamente nos computadores dos clientes que já clicaram anteriormente em um banner do Clickwise.

                
Se uma compra for feita diretamente da página sem passar previamente pelo nosso redirecionamento, o pixel não funcionará



                
O pixel
                    
                


                
<img width="1" height="1" src="https://r.clickwise.net/t/{campaign-id}/{action-code}/{order-id}?total-cost={total-cost}&currency={currency}"/>


                
URL de rastreamento
                    
                

                

                
Possíveis configurações:

                
Host: r.clickwise.net
Path Params: /t/{campaign}/{action-code}/{order-id}

                

                    
                        

                            Field
                            Type
                            Description
                        

                    
                    
                        

                            campaign-idobrigatório
                            String
                            ID único da Campanhha cadastrado na platforma Clickwise
                        

                        

                            action-codeobrigatório
                            String
                            Código de ação válido ativo na campanha
                        

                        

                            order-idobrigatório
                            String
                            Número unico do pedido(ID do pedido) 
                        

                    
                

                
Query Params:
                    
                

                

                
Todos os parâmetros de consulta aceitos pelo URL de rastreamento. (param1={param1}¶m2={param2})

                

                    
                        

                            Field
                            Type
                            Description
                        

                    
                    
                        

                            total-cost% obrigatório 
                            Float
                            Valor Total da transação sem o frete
                        

                        

                            currencyopcional
                            String
                            Código de moeda de 3 caracteres
